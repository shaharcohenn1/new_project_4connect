זה פרויקט שבניתי שבו יצרתי משחק "ארבע בשורה" רשת עם צד שרת ומסד נתונים. המטרה הייתה ללמוד איך עובדים עם Node.js, Express ו־MySQL, ולחבר ביניהם בשביל לנהל משחקים בזמן אמת.

מה עשינו בפרויקט

מסד נתונים (MySQL):

יצרנו מסד נתונים בשם game_db.

יש שתי טבלאות עיקריות:

games – שומרת מידע על מצב המשחק (לוח, צבע תור, סטטוס).

players – שומרת מידע על השחקנים שמשתתפים במשחק.

הוספנו קשרים בין טבלאות כדי שמחיקה של משחק תמחוק גם את השחקנים שלו.

שרת (Node.js + Express):

השרת רץ על פורט 4000.

מטפל ביצירת משחק חדש, הצטרפות למשחק קיים, שמירת מהלכים ועדכון מצב המשחק במסד הנתונים.

השתמשנו ב־WebSockets כדי לאפשר עדכון בזמן אמת בין השחקנים.

צד לקוח (Frontend פשוט):

ממשק בסיסי ב־HTML + CSS + JS.

לוח המשחק מוצג כרשת, וכל שחקן בוחר עמודה כדי להפיל דיסקית.

המידע נשלח לשרת והעדכון חוזר לשני השחקנים.

איך מריצים את הפרויקט

מתקינים חבילות בצד השרת:

cd server npm install npm start

פותחים את הדפדפן בכתובת:

http://localhost:4000

יוצרים משחק חדש או מצטרפים עם קוד משחק קיים – ואז מתחילים לשחק
